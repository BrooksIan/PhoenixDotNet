Apache Phoenix Query Interface

Quick Start

  • Enter SQL queries in the text area below
  • Click "Execute Query" to run SELECT statements
  • Click "Execute Command" for CREATE, INSERT, UPDATE, DELETE
  • Use "List Tables" to see all available tables
  • Use "List Views" to see all Phoenix views (for HBase-native tables)

Important: UPPERCASE Table Names

  • Phoenix converts unquoted identifiers to UPPERCASE
  • When you create: CREATE TABLE users (...) ‚Üí Query as: SELECT * FROM USERS
  • Always use UPPERCASE table names when querying (e.g., USERS, TABLE_NAME)
  • Query tables directly instead of relying on the list endpoint
